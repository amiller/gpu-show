gpu-show / frozen gallery

100 animated 3D scenes, each invented and coded in one shot by a local 27B LLM (Qwen3.5 27B) writing WebGL from scratch. The model emits a fragment-shader body and a scene-builder function; the harness owns the rest (primitives, camera, lighting). Nothing here is running an LLM — these are replays of saved programs. live-generation source on github.
